Getting Started - Installation Instructions:
* See INSTALL file for requirements to install and use this program
* Go to the main branch of this project
* Download the zip file for this project by pressing the "Code" button and clicking "Download ZIP"
* Move the downloaded ZIP folder and rename it, if desired (just be sure you know the folder's name and location--this will be important for setting up and uninstalling the program)
* To start setup, unzip the folder; refer to the INSTALL file for further instructions
